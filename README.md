MySDLGame2 (制作進行中)
C++と最新のマルチメディアライブラリ SDL3 を使用した、2Dアクションゲームの開発プロジェクトです。既存のゲームエンジンに依存せず、ゲームが動く根幹の仕組みを自力で制御・理解することを目的として制作しています。

概要
・開発環境: Windows 11, Visual Studio 2022
・言語: C++
・使用ライブラリ: SDL3, SDL3_image

実装済みの主要機能
・ステートパターンを用いたキャラクター制御: PlayerState クラスを基底とし、Idle, Run, Jump, Airborne の各状態をクラスとして独立させています。これにより、複雑なアクションの遷移を整理し、拡張性の高い設計を行っています。
・外部ファイル（CSV）によるパラメータ管理: プレイヤーの移動速度やジャンプ力などの数値を PlayerParams.csv から読み込む仕組みを実装しています。プログラムを書き換えずに挙動を調整できる、データ駆動型の設計を意識しています。
・物理演算・衝突判定: AABB（Axis-Aligned Bounding Box）を用いた矩形判定による衝突検知を自作しています。
