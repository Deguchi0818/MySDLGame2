MySDLGame2 (制作進行中)
C++と最新のマルチメディアライブラリ SDL3 を使用した、2Dアクションゲームの開発プロジェクトです。既存のゲームエンジンに依存せず、ゲームが動く根幹の仕組みを自力で制御・理解することを目的として制作しています。
